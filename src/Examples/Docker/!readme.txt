This example shows the following:

1. How to initialize Scabra without using Scabra.Hosting types.
2. How to host Scabra in Docker. 

To run the Docker example do the following:

1. create 'scabra' network with 'create-network.cmd';
2. build 'server' and 'client' images with 'build-images.cmd';
3. run 'server' and 'client' containers with 'run-containers.cmd'.
